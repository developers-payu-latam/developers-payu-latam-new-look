El producto Tokenización te permite almacenar los datos de las tarjetas de crédito de tus clientes de forma segura, por medio de la creación de un token con el cuál podrás realizar cobros periódicamente o implementar la funcionalidad de pago en 1 Clic. Siguiendo así los estándares de seguridad en la manipulación de los datos de tarjetas de crédito PCI DSS (Payment Card Industry Data Security Standard).

Para el uso de las funcionalidades del producto de Tokenización desde el API, debes tener habilitado este servicio en tu cuenta PayU, si deseas adquirirlo comunícate con sac@payulatam.com.
